# ANTS Bandcamp Uploader (Non-Technical Instructions)

## What you send to a member
You will send them a single ZIP file for their computer:
- **Windows:** a ZIP that contains `ANTS Bandcamp Uploader.exe`
- **Mac:** a ZIP that contains `ANTS Bandcamp Uploader.app`

They just unzip it and double-click the app.

## If Windows shows a warning
- Click **More info** → **Run anyway**.

## If Mac blocks the app (Gatekeeper)
- Right-click the app → **Open** → **Open**.
- This is normal for unsigned apps.

## What they do inside the app
1. Click **Browse** and pick the Export folder (or Export ZIP).
2. Type the album title.
3. Click **Start Upload**.
4. The browser opens and uploads the tracks.

That’s it.
