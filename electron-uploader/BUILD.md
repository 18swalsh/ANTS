# Build Outputs (Unsigned)

## Windows
From `electron-uploader/`:
- `npm install`
- `npm run pack:win`
Result: `electron-uploader\dist\ANTS Bandcamp Uploader-*.zip`

## macOS
From `electron-uploader/` on a Mac:
- `npm install`
- `npm run pack:mac`
Result: `electron-uploader/dist/ANTS Bandcamp Uploader-*.zip`

Send the ZIP to the member. They unzip and run the app.
