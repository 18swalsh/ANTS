# Bandcamp Uploader (Playwright)

This script automates Bandcamp album uploads using the `bandcamp_upload.csv` generated by the Apps Script.

## Setup
1. Install Node.js (18+ recommended).
2. In this folder:
   ```
   npm install
   ```
3. Copy `settings.example.json` to `settings.json` and tweak if needed.

## Run
```
npm run upload
```

## Easiest Way (Windows)
Double-click `run-uploader.cmd`.

## Standalone EXE (No Node Needed)
Build once and share the EXE and GUI launcher.
See `STANDALONE.md`.

You will be prompted for:
- Export folder path (contains `bandcamp_upload.csv` and audio files)
- Album title (custom per month)
- Album art file path (optional)
- Album edit URL (defaults to https://americasnexttopsong.bandcamp.com/edit_album)
  
You can also provide a `.zip` export file; it will be extracted automatically.

If you leave the URL blank, the script opens Bandcamp and waits for you to log in and navigate to the album edit page manually.

## Notes
- The script stops short of publishing so you can review and publish manually.
- If Bandcamp UI selectors change, update `settings.json` under `selectors`.
